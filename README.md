# Aabo

Rwanda's safest motorcycle riding platform app.

## Overview

Aabo is a mobile application designed to provide a safe and reliable motorcycle riding experience in Rwanda. With a focus on safety, convenience, and user satisfaction, Aabo connects riders or passengers with trusted motorcycle drivers, ensuring a seamless and secure journey.

## Features

- **Safe Rides**: Verified drivers and safety-first policies.
- **User-Friendly Interface**: Simple and intuitive design for all users.
- **Real-Time Tracking**: Track your ride in real-time for added security.
- **Affordable Pricing**: Transparent and competitive pricing.
- **Multi-Platform Support**: Available on Android, iOS, and web platforms.

## Installation

To get started with Aabo, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/aabo.git
   ```
2. Navigate to the project directory:
   ```bash
   cd aabo
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```
